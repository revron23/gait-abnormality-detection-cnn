## Dataset Information

The dataset used in this project consists of Ground Reaction Force (GRF) measurements and related metadata used for gait abnormality classification.

### Dataset Details
- Format: Excel / CSV
- Size: ~100 MB
- Features include:
  - Vertical, mediolateral, and anteroposterior GRF
  - Center of Pressure (CoP)
  - Subject metadata (age, gender, body metrics)
